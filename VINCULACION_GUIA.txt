PASO 1: AGREGAR LINKS EN EL MENÚ PRINCIPAL (header.html)
═══════════════════════════════════════════════════════════════════════════════

Busca en tu archivo header.html la sección de navegación (probablemente <nav> o <ul class="nav-menu">)
y agrega estos dos links:

OPCIÓN A - Si usas estructura HTML básica:
──────────────────────────────────────────────────────────────────────────────
<nav class="navbar">
    <ul class="nav-menu">
        <li><a href="index.html">Inicio</a></li>
        <li><a href="aboutus.html">Sobre Nosotros</a></li>
        <li><a href="services.html">Servicios</a></li>
        
        <!-- AGREGAR ESTOS DOS LINKS -->
        <li><a href="prices.html"><i class="fas fa-tag"></i> Precios</a></li>
        <li><a href="calculator.html"><i class="fas fa-calculator"></i> Calculadora</a></li>
        
        <li><a href="contact.html">Contacto</a></li>
    </ul>
</nav>
──────────────────────────────────────────────────────────────────────────────

OPCIÓN B - Si usas versión con más estilos:
──────────────────────────────────────────────────────────────────────────────
<div class="nav-item">
    <a href="prices.html" class="nav-link">
        <i class="fas fa-tag"></i>
        <span>Precios</span>
    </a>
</div>

<div class="nav-item">
    <a href="calculator.html" class="nav-link">
        <i class="fas fa-calculator"></i>
        <span>Calculadora</span>

</div>

PASO 4: INTEGRAR WHATSAPP EN LOS BOTONES CTA
═══════════════════════════════════════════════════════════════════════════════

Los botones "Contratar por WhatsApp" en ambas páginas pueden enviarse a tu 
número de WhatsApp con el presupuesto. Para hacerlo funcionar:

REEMPLAZAR EN PRICES.HTML (línea ~450):
──────────────────────────────────────────────────────────────────────────────
CAMBIAR:
    <button class="plan-cta"><i class="fas fa-whatsapp"></i> Reservar Plan</button>

POR:
    <a href="https://wa.me/5491234567890?text=Hola!%20Me%20interesa%20el%20Plan%20de%20Mantenimiento"
       class="plan-cta" target="_blank">
        <i class="fas fa-whatsapp"></i> Reservar Plan
    </a>

NOTA: Reemplaza "5491234567890" con tu número de WhatsApp (código de país + número)
──────────────────────────────────────────────────────────────────────────────

REEMPLAZAR EN CALCULATOR.HTML (línea ~650):
──────────────────────────────────────────────────────────────────────────────
CAMBIAR:
    <button class="btn-calculate">
        <i class="fas fa-whatsapp"></i> Contratar por WhatsApp
    </button>

POR:
    <a href="https://wa.me/5491234567890?text=Presupuesto%20generado%20en%20calculadora"
       class="btn-calculate" target="_blank">
        <i class="fas fa-whatsapp"></i> Contratar por WhatsApp
    </a>
──────────────────────────────────────────────────────────────────────────────

ALTERNATIVA: VERSIÓN MEJORADA CON JAVASCRIPT
──────────────────────────────────────────────────────────────────────────────
Si quieres enviar el presupuesto calculado en el mensaje de WhatsApp:

En calculator.html, modifica el botón así:

<button class="btn-calculate" onclick="sendToWhatsApp()">
    <i class="fas fa-whatsapp"></i> Contratar por WhatsApp
</button>

Y agrega este script al final del archivo (antes de </body>):

<script>
function sendToWhatsApp() {
    const phoneNumber = '5491234567890'; // TU NÚMERO
    const finalPrice = document.getElementById('finalPrice').textContent;
    const frequency = document.getElementById('detailFrequency').textContent;
    const service = document.getElementById('detailService').textContent;
    
    const message = `Hola! Me interesa contratar: ${service}. Presupuesto: ${finalPrice} (${frequency})`;
    const encodedMessage = encodeURIComponent(message);
    const whatsappUrl = `https://wa.me/${phoneNumber}?text=${encodedMessage}`;
    
    window.open(whatsappUrl, '_blank');
}
</script>
──────────────────────────────────────────────────────────────────────────────

PASO 8: AGREGAR ANALYTICS (Google Analytics) - Opcional
═══════════════════════════════════════════════════════════════════════════════

Para rastrear cuántas personas visitan estas páginas, agrega Google Analytics:

Agrega esto antes de </head> en ambas páginas:

<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());

  gtag('config', 'GA_MEASUREMENT_ID');
</script>

Reemplaza GA_MEASUREMENT_ID con tu ID de Google Analytics.
